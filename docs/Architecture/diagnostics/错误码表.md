## 错误码分段规则

```
E + 分类编号(1-9) + 顺序号(00-99)
```

| 分段 | 含义 |
|------|------|
| E1xxx | IR 生成层错误 |
| E10xx | 节点/符号基础错误 |
| E11xx | 函数定义相关错误 |
| E12xx | 函数调用相关错误 |
| E13xx | 全局/初始化相关错误 |
| E14xx | 变量定义相关错误 |

## 错误详情

### E1000 — 不支持的 AST 节点类型

**源码**：`IRGenerator.cpp:149`

```cpp
report_ir_error("E1000", node != nullptr ? node->line_no : -1,
    "节点分发", "不支持的AST节点类型(%d)", (int) node->node_type);
```

**触发条件**：遇到未知类型的 AST 节点类型

**输出示例**：
```
IR错误[E1000] 第3行 节点分发: 不支持的AST节点类型(42)
```

## 维护说明

1. **新增错误码**：在 `IRGenerator.cpp` 或 `Module.cpp` 中使用 `report_ir_error()` 或 `Status::Error()` 时分配新码
2. **更新文档**：每次添加/修改错误码后，同步更新本表格
3. **代码规范**：优先使用 `report_ir_error()` 报告 IR 层错误，确保输出格式统一
4. **行号注意**：直接使用 `Status::Error()` 时无法传入行号，会输出 `未知行`
